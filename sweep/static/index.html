PAD steering sweep

Layer-range steering experiments: vectors, response sweeps, PAD scoring, and rankings analysis.

Overview

This tool automates PAD (pleasure–arousal–dominance) steering experiments over layer ranges in a causal language model. You train steering vectors from contrastive prompt pairs, sweep steering multipliers on fixed scenarios, score generations with a PAD classifier (and optional LLM judge), then compare layer spans via rankings and heatmaps.

Repository layout

  • steering_vectors/*.pt — trained steering vectors (committed for reproducibility)
  • sweep/results/responses/ — generation JSONL per range and run
  • sweep/results/scoring/ — scored rows
  • sweep/results/analysis/ — aggregated rankings JSON/CSV
  • sweep/pairs/{dimension}.json — training contrast pairs
  • sweep/scenarios.json — evaluation prompts

Quick start

From the repository root (with dependencies installed; see README.md):

python sweep/server.py

Open http://127.0.0.1:8765. Use Results to browse rankings and charts from committed artifacts, or run workflow tabs to extract, generate, score, and analyze.

CLI (same backend)

python sweep/extract.py --dimension valence --start 8 --end 14 --num-layers 32
python -m sweep.pipeline plan --preset full32 --dimensions valence
python -m sweep.pipeline run --preset full32 --state sweep/results/pad_pipeline_state.json
python sweep/generate.py --vector … --range-id valence_8_14 --dimension valence …
python sweep/score.py --input …
python sweep/analyze.py

Root scripts

  • load_model.py — configure and load the base causal LM
  • pad_config.py, emotion_classifier.py — PAD scoring classifier
  • training/train.py — retrain classifier (GoEmotions → emotion_model_final/)
  • extract_*_pairs.py — legacy pair export helpers
  • test_steering.py — single-vector generation smoke test
  • orthogonality.py — compare three .pt vectors (angles / Gram matrix)
